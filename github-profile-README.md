## About Me

Firmware engineer with **5+ years** building production-grade embedded systems for medical devices, industrial IoT platforms, and smart wearables. I work across the full stack — from bare-metal MCU bring-up and RTOS architecture to BLE protocol design, analog front-end circuits, and cloud connectivity.

- 🏥 Currently building safety-critical firmware at **Caropet Technologies** (medical devices)
- ⚡ Achieved **sub-500nA sleep current** on BGM220-based BLE wearables in production
- 🌐 Deployed firmware across **2500+ devices** in the field
- 📡 Running **5 active LoRa hubs** for BeetleGuard environmental monitoring
